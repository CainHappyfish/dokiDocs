# 链接

用来跳转到指定链接上。

## 基础用法

```vue
<doki-link type="primary">primary</doki-link>
<doki-link type="success">success</doki-link>
<doki-link type="info">info</doki-link>
<doki-link type="warning">warning</doki-link>
<doki-link type="danger">danger</doki-link>
```

## API

|   API    |  类型  |     用途     |
| :------: | :----: | :----------: |
|   type   | string | 指定按钮种类 |
| disabled |   -    |   禁用标签   |
