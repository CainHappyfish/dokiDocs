# 单选框

用于单项选择。

## 基础用法

```vue
<doki-radio value="enabled1" label="enabled1" name="able"/>
<doki-radio value="enabled2" label="enabled2" name="able"/>
<doki-radio value="disabled" label="disabled" name="able" disabled=""/>
```

## API

|   API    |       类型       |     用途     |
| :------: | :--------------: | :----------: |
|  value   | string \| number | 选中状态的值 |
|  label   |      string      |  复选框标签  |
| disabled |        -         |     禁用     |
